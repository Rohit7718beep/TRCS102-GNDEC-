# Daily Training Diary – Day 2

Today was the second day of my training, and I learned about creating forms in HTML and collecting user input through different form elements.

I started by learning how to create an HTML form using the `<form>` tag and how to organize input fields using the `<label>` tag for better accessibility and readability.

During the session, I practiced using different types of input fields, including:

* Text input for entering names using `type="text"`.
* Email input using `type="email"` for validating email addresses.
* Password fields using `type="password"` to hide user input.
* Date input using `type="date"` for selecting dates of birth.
* File upload fields using `type="file"` to allow users to upload files.
* Time input using `type="time"` for selecting a specific time.
* Checkbox inputs for selecting multiple options.
* Radio buttons for selecting a single option such as gender.
* Dropdown menus using the `<select>` and `<option>` tags for selecting cities.

I also learned the importance of using the `name` attribute in radio buttons to ensure that only one option can be selected at a time.

Today's practical session helped me understand how forms are used to collect user information on websites such as registration pages, login pages, surveys, and feedback forms.
Overall, Day 2 was very informative and improved my understanding of HTML forms and user input handling.

# Daily Training Diary – Day 3

Today was the third day of my training, and I was introduced to the basics of CSS and JavaScript, which are essential technologies for creating interactive and visually appealing websites.

I learned how to connect an external CSS file to an HTML document using the `<link>` tag and how to connect an external JavaScript file using the `<script>` tag. This helped me understand the separation of structure, design, and functionality in web development.

In CSS, I explored the concept of styling web pages and learned how CSS can be used to modify the appearance of HTML elements by changing properties such as colors, spacing, alignment, and layout. I also practiced using IDs and containers for applying styles to specific elements on a webpage.

In JavaScript, I learned the basics of programming and how to add interactivity to websites. The topics covered included:

* Displaying output in the browser console using `console.log()`.
* Showing popup messages using `alert()`.
* Declaring variables using `let`, `const`, and `var`.
* Updating variable values and understanding the difference between mutable and constant variables.
* Learning different data types such as strings, numbers, booleans, `null`, and `undefined`.
* Using the `typeof` operator to check the type of a variable.
* Understanding naming conventions such as snake case and camel case.

I also practiced different types of operators in JavaScript, including:

* Arithmetic operators (`+`, `-`, `*`, `/`, `%`)
* Assignment operators (`=`, `+=`, `-=`, etc.)
* Logical operators (`&&`, `||`, `!`)
* Comparison operators (`>`, `<`, `!=`, `==`, `===`)
* Increment and decrement operators (`++`, `--`)

Additionally, I learned the difference between loose equality (`==`) and strict equality (`===`) and understood how JavaScript handles type comparison.

Overall, Day 3 provided a strong foundation in web styling and programming concepts, helping me understand how websites become visually attractive and interactive for users.
